## Installation

1. Open terminal, then run:

    ```shell
    pip install tokysnatcher
    ```

    or

    ```shell
    pip install git+https://github.com/rahaaatul/TokySnatcher.git
    ```

2. Run TokySnatcher

    ```sh
    tokysnatcher
    ```

## Usage

- In command line invoke `-h` for help:

    ```shell
    tokysnatcher -h
    ```

- Invoke `-d` or `--directory` to add a custom directory

    ```shell
    tokysnatcher -d "C:\Users\User\Music"
    ```

> [!NOTE]
>
> - By default, TokySnatcher saves audiobooks to your system's Music folder in an "Audiobooks" subfolder (e.g., `C:\Users\User\Music\Audiobooks\` on Windows)
> - Use `-d` or `--directory` to specify a custom location
>
